# Todo App Overview
## Purpose
A todo application evolving from CLI to full-stack web with multi-user support.
## Current Phase
Phase II: Full-Stack Web Application
## Tech Stack
- Frontend: Next.js 16+, TypeScript, Tailwind CSS
- Backend: FastAPI + SQLModel
- DB: Neon Serverless PostgreSQL
- Auth: Better Auth (JWT)
## Features
- [ ] Task CRUD
- [ ] User Authentication
- [ ] Priorities & Tags
- [ ] Search & Filter
- [ ] Sort Tasks
